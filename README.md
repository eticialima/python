# Estudos Python Cursos 

## Este projeto foi feito com:

* [Python 3.10.2](https://www.python.org/) 

## Como rodar o projeto?

* Crie um virtualenv com Python 3.
* Ative o virtualenv. 

``` 
python -m venv .venv
source .venv/bin/activate 
python "nome do arquivo .py"
```
